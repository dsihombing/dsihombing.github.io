Pengajaran

SPK Decision Support Systems

Sistem Pendukung Keputusan

Mata kuliah ini memberikan pengenalan komprehensif mengenai sistem pendukung keputusan (SPK), mencakup konsep dasar, arsitektur, dan metodologi untuk meningkatkan kualitas pengambilan keputusan di organisasi.

Mahasiswa mempelajari teknik pengambilan keputusan multikriteria (MCDM) seperti AHP, TOPSIS, dan SAW, serta pendekatan berbasis model dan berbasis data untuk mendukung perencanaan dan strategi organisasi.

  • Konsep dan arsitektur SPK.
  • Metode MCDM (AHP, TOPSIS, SAW).
  • SPK berbasis model dan berbasis data.
  • Analisis keputusan dan visualisasi.
ML Machine Learning

Machine Learning

Mata kuliah ini memperkenalkan konsep dasar machine learning dan teknik praktis untuk menggali insight dari data melalui pemodelan dan analisis.

Mahasiswa memperoleh pengalaman langsung dalam pengolahan data, pra-pemrosesan, analisis data eksploratori (EDA), serta pengembangan model prediktif dengan berbagai algoritma machine learning.

  • Pengolahan dan pra-pemrosesan data.
  • Analisis data eksploratori (EDA).
  • Pemodelan prediktif dan algoritma ML.
  • Visualisasi data dan storytelling.
DX Digital Transformation

Transformasi Digital

Mata kuliah ini mengkaji dimensi strategis transformasi digital di organisasi modern, serta bagaimana teknologi baru dan sistem informasi mendorong inovasi dan keunggulan bersaing.

Mahasiswa menganalisis kerangka kerja transformasi digital, inovasi model bisnis, pengembangan strategi digital, pengambilan keputusan fuzzy pada lingkungan yang tidak pasti, dan praktik manajemen perubahan di era digital.

  • Strategi digital dan inovasi model bisnis.
  • Pengambilan keputusan fuzzy.
  • Kerangka dan metodologi transformasi digital.
  • Perubahan organisasi di era digital.
