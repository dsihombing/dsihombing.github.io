Course Overview

This comprehensive machine learning course covers fundamental to advanced ML algorithms and techniques. Students will learn supervised and unsupervised learning methods, from linear regression to deep neural networks, with extensive hands-on coding in Python.

Through practical projects using real-world datasets, you'll develop the skills to build, train, evaluate, and deploy machine learning models for various applications in business, science, and technology.

Learning Outcomes

  • Understand ML fundamentals: supervised, unsupervised, and reinforcement learning
  • Implement regression and classification algorithms from scratch
  • Master ensemble methods: Random Forest, Gradient Boosting, XGBoost
  • Build and train neural networks using TensorFlow and Keras
  • Apply feature engineering, selection, and dimensionality reduction
  • Evaluate model performance and prevent overfitting
  • Deploy ML models for production use

Course Topics

Week 1-2: ML Foundations

Introduction to ML, Python libraries (NumPy, Pandas), data preprocessing, and exploratory analysis

Week 3-4: Supervised Learning I

Linear regression, logistic regression, and polynomial features

Week 5-6: Supervised Learning II

Decision trees, K-NN, Naive Bayes, and SVM

Week 7-8: Ensemble Methods

Bagging, boosting, Random Forest, and XGBoost

Week 9-10: Unsupervised Learning

K-Means clustering, hierarchical clustering, DBSCAN, and PCA

Week 11-12: Neural Networks

Perceptrons, backpropagation, CNNs, and RNNs with TensorFlow

Week 13: Model Optimization

Hyperparameter tuning, cross-validation, and regularization

Week 14: Final Project

End-to-end ML project with real dataset and deployment
